# 🔒 Login - Sign up - Sign in

This is a free basic template with authentication, token validation and encryption.

📚 Technologies: 
  - NodeJS
  - Mongoose
  - Express
  - Bcrypt
  - Hapi/Joi

#### 🏗️ How to run
```
npm install
npm run dev
```
✔️ You are free to use it as your want.
